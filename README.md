Airbnb Berlin vs. Munich: Pricing and Availability Analysis
Analyzed Airbnb listing data across Berlin and Munich to identify pricing drivers, availability trends, and market differences between the two cities.
Tools Used

Python (Pandas, NumPy) for data cleaning and aggregation
NLP for text analysis of listing descriptions
Power BI for dashboard visualization

Key Findings

Identified primary pricing drivers across both markets
Analyzed availability patterns and seasonal trends
Compared listing characteristics between Berlin and Munich
